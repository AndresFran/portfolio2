Acerca de mi

Seguir caminando,
seguir avanzando

Nací en la provincia de San Juan en el año 1979. He tenido una infancia feliz rodeada de amigos de barrio, los cuales aun conservo.

Desde que superé una grave enfermedad que tuve en el 2013, leucemia mieloide aguda, me prometí que ya no sería el mismo. A partir de eso sucedieron muchas cosas. Me puse de novio a los 34 años y me compré una moto a los 36 para poder viajar. Formé una banda de rock a los 37 (aún vigente) y a los 38 años me casé. Tuve una hermosa hija a los 40, después de haber viajado con mi esposa por casi todo Europa y a los 41 comencé a estudiar Programación de forma autodidacta.

En medio de todo eso, seguí trabajando.

Siempre busco desafíos. Me gustan las aventuras que me permiten crecer, como la de ser padre. Programar se ha convertido en una aventura fascinante, donde día a día voy descubriendo nuevas cosas. Mi objetivo es seguir caminando... seguir avanzando.

Mi trabajo

¿Qué hice en este tiempo?

Mi primer Portfolio

Portfolio realizado con HTML y CSS. Fue mi primer gran experiencia con el diseño web. Puede parecer muy rudimentario pero fue mi primer gran proyecto.

Sitio Web Inmobiliaria

Página web para una inmobiliaria. Realicé el diseño y desarrollo web utilizando HTML, CSS y JavaScript. Además colaboré en la parte de base de datos con MySQL y backend a través de PHP.

Sitio Web banda de Rock

AZAR es mi banda de rock, dónde canto, toco la guitarra y compongo parte de las canciones. La página está desarrollada y diseñada enteramente por mi. La hice con HTML, CSS y JavaScript, lo que permite poder escuchar nuestra música mediante un reproductor.

Sitio Web Fábrica

Diseñé una página Web estática para una empresa de San Juan la cual quería mostrar los productos que realizaban. En un futuro tienen intenciones de ampliar este sitio a uno dinámico dónde puedan cargar imágenes y tener un blog sobre políticas medioambientales.

Sitio Web empresa de servicios IT

Diseñé una página web para esta empresa que se dedica a brindar soluciones informáticas, entre otras cosas. La página está desarrollada en HTML, CSS y JavaScript. La parte de base de datos y BackEnd se encargaron ellos mismos.
